# Keyboard Shortcuts

NaijaStay includes convenient keyboard shortcuts to help you navigate the application more efficiently.

## Available Shortcuts

### Admin Dashboard Navigation
- **Ctrl + Shift + A** - Navigate directly to the Admin Dashboard
- **Ctrl + Shift + ?** - Show this help dialog

## How to Use

1. Make sure you're logged in with an admin account
2. Press the key combination for the desired action
3. The application will automatically navigate to the requested page

## Supported Browsers

These keyboard shortcuts work in all modern browsers:
- Chrome
- Firefox
- Safari
- Edge

## Notes

- Keyboard shortcuts are only active when the application is in focus
- The Ctrl key on Windows/Linux is equivalent to the Cmd key on macOS
- Shortcuts will not interfere with browser or system-level shortcuts
- A toast notification will appear when navigating using shortcuts

## Troubleshooting

If keyboard shortcuts are not working:

1. Ensure you're logged in as an admin user
2. Check that the application window is in focus
3. Verify your browser supports JavaScript keyboard events
4. Try refreshing the page
