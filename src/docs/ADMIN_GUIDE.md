# Admin Dashboard Guide

This guide explains all the features and functionalities available in the NaijaStay Admin Dashboard.

## Overview

The Admin Dashboard provides comprehensive tools for managing hotels, rooms, bookings, users, and communications. It offers insights through analytics and enables efficient administration of the entire platform.

## Accessing the Admin Dashboard

1. Log in with an admin account (email: victorralph407@gmail.com)
2. Navigate to `/admin` or click on the "Admin Dashboard" link in the user menu

## Dashboard Features

### 1. Statistics Overview

The dashboard homepage displays key metrics:
- Total Hotels
- Total Rooms
- Bookings Count
- User Count
- Revenue Generated

### 2. Hotel Management

Manage all hotels in the system:
- View all hotels in a list format
- Edit hotel details
- Delete hotels
- Add new hotels (button in top right corner)

### 3. Room Management

Manage all rooms across hotels:
- View all rooms with pricing information
- Edit room details
- Delete rooms
- Add new rooms

### 4. Booking Management

Handle all bookings in the system:
- View all bookings with status indicators
- Confirm pending bookings
- Cancel bookings
- View booking details

### 5. User Management

Manage registered users:
- View all users with their roles
- Change user roles (User/Admin)
- View user registration dates

### 6. Analytics Dashboard

Gain insights into business performance:
- Booking trends visualization
- Revenue overview charts
- Popular hotels ranking
- Customer feedback metrics

### 7. Communication Management

Handle all customer communications:

#### Newsletter Management
- Send newsletters to subscribers
- View subscriber statistics
- See recent subscribers

#### Contact Messages
- View all contact form submissions
- Mark messages as read
- Reply to customer inquiries
- Track message status (unread, read, replied)

### 8. Image Management

Upload images for hotels and rooms:
- Select multiple images at once
- Automatic upload to Firebase Storage
- Images are optimized for web use

## Best Practices

### User Management
- Only grant admin privileges to trusted personnel
- Regularly review user accounts
- Monitor user activity through bookings

### Content Management
- Keep hotel information up-to-date
- Ensure room descriptions are accurate
- Maintain current pricing information

### Communication
- Respond to customer inquiries promptly
- Send regular newsletters to engage users
- Use analytics to improve customer experience

### Security
- Use strong passwords for admin accounts
- Regularly review security settings
- Monitor login activity

## Troubleshooting

### Common Issues

1. **Permission Errors**
   - Ensure you're logged in with an admin account
   - Check Firebase security rules

2. **Image Upload Failures**
   - Verify file size (max 10MB)
   - Check file format (PNG, JPG, GIF)
   - Ensure stable internet connection

3. **Data Not Loading**
   - Refresh the page
   - Check Firebase connection
   - Verify database rules

### Getting Help

For technical issues, contact the development team or refer to the Firebase Setup Guide in the application.
